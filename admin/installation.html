Installation

Primeiramente quero agradecer por adquirir nosso template, estamos felizes com isso. Antes de continuarmos verifique se seu computador tem instalado o Node.js e o npm. Abra o terminal e execute o comando abaixo:

node -v && npm -v

Caso não tenha instalado estes softwares acesse este link e faça a instalação do Node.js.

Bower

O gerenciador de pacotes Bower é utilizado para baixar todas as dependências do frontend, como jQuery, Bootstrap, Parsley entre diversas outras bibliotecas contidas no arquivo bower.json. Faça a instalação do Bower globalmente através do comando:

npm install -g bower

Grunt

O executador de tarefas Grunt é responsável por todas as tarefas pré-programadas contidas no arquivo Gruntfile.js. São diversas tarefas como minificação de arquivos, qualidade de código, versionamento, entre outras. Faça a instalação do Grunt globalmente através do comando:

npm install -g grunt-cli

Project Dependencies

Feita a instalação do softwares necessários precisamos agora instalar todas as dependências do projeto, execute o comando abaixo:

npm install && bower install

Run Project

Se tudo ocorrer bem basta agora executar o comando a seguir e sua aplicação irá abrir no seu navegador padrão o endereço http://localhost:9000/:

grunt serve

Caso obtenha algum problema acesse a página resolvendo problemas.
